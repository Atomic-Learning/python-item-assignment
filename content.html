For some collections in Python it is possible to change the value of an item at a specific index. Common examples include lists and Numpy arrays. Here, we will use lists as an example:

Changing a Single Item

To assign a new value to an item at a specific index, we write the name of the collection, followed by square brackets containing the index, and then an equals sign with the new value on the right:

my_list = [1, 2, 3, 4] print(my_list) my_list[1] = 10 * 2 # The value on the right can be an expression print(my_list) my_list[-1] = 40 # We can also use negative indexing print(my_list)
